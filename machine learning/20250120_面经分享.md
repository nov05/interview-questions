### ML Coding
```python
# precision computation (interview version)
def precision(targets, predictions, threshold):
    TP = 0
    FP = 0
    TN = 0
    FN = 0

    for pred, tar in zip(predictions, targets):
        if pred > threshold:
            if tar == pred:
                TP += 1
            else:
                FP += 1
        else:
            if tar == pred:
                TN += 1
            else:
                FN += 1

    precision = TP / (TP + FP)
    return precision
````

> 注：逻辑上这里更标准应是 `tar == 1/0`，而不是 `tar == pred`

## C4：Google SWE PhD Intern

### 题目 1（terrain + fountains）

**题意：**

* terrain array = height
* fountains array = positions
* 从 fountain 向左右扩展：

  * 如果比当前值小，则标记为 1
* 输出 bitmask

```text
terrain = [2, 1, 1, 3, 2, 1, 1]
fountains = [0, 3]
return = [1, 1, 1, 0, 1, 1, 1]
```

**解法：**

* O(kN) per fountain（暴力）
* O(N) monotonic stack（最优）
